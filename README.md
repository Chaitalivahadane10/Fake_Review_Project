# 🕵️ Fake Review Detection Project

## 📌 Description
This project focuses on detecting fake reviews using **Machine Learning** and **Natural Language Processing (NLP)** techniques. It helps identify whether a review is genuine or fake based on text analysis.

---

## 🚀 Features
- Data Preprocessing and Cleaning  
- Text Analysis using NLP  
- Machine Learning Model Training  
- Fake Review Prediction System  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Natural Language Processing (NLP)  

---

## 📂 Project Structure
- `app.py` → Main application  
- `main.py` → Model training & logic  
- `reviews.csv` → Dataset  

---

## 🎯 Objective
To build a system that can automatically detect fake reviews and improve trust in online platforms.
